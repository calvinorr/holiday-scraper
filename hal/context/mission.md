# Mission

## Project: Holiday Scraper

A Next.js application that scrapes holiday deals from travel websites and presents them to users.

## Purpose

Enable users to find and compare holiday deals across multiple travel providers. The app will:
- Scrape holiday listings from travel websites
- Store deals in a Turso database
- Use AI (Gemini) to enhance/categorize deal information
- Present deals in a clean, searchable interface

## Success Criteria

- Reliable scraping of holiday deals
- Fast search and filtering
- Clean, responsive UI
- Data persistence with Turso/Drizzle

## Tech Stack

- **Frontend**: Next.js 16 (App Router), Tailwind CSS
- **Database**: Turso (libsql) with Drizzle ORM
- **Scraping**: Cheerio
- **AI**: Google Generative AI (Gemini)
- **Icons**: Lucide React
