# Visual Standards

## Design System

### Colors (Dark Mode First)
- Background: `zinc-950` / `black`
- Card: `zinc-900`
- Border: `zinc-800`
- Text Primary: `zinc-50`
- Text Muted: `zinc-400`
- Accent: `emerald-500` (deals/savings)
- Warning: `amber-500` (price alerts)

### Typography
- Font: System UI / Geist
- Headings: `font-semibold tracking-tight`
- Body: `text-zinc-400 leading-relaxed`

### Components
- Cards: `rounded-xl border border-zinc-800 bg-zinc-900`
- Buttons: `rounded-full px-4 py-2`
- Inputs: `rounded-lg border border-zinc-700 bg-zinc-800`

### Layout
- Max width: `max-w-6xl`
- Padding: `px-4 sm:px-6`
- Grid: `grid-cols-1 md:grid-cols-2 lg:grid-cols-3`

## Component Patterns

### Deal Card
- Image at top (aspect-video)
- Destination & dates
- Price with savings badge
- Provider logo
- Quick action buttons

### Search/Filter Bar
- Sticky header
- Destination autocomplete
- Date range picker
- Price range slider
- Provider filter chips
